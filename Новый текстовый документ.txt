🚀 Запуск (шпаргалка на стену)
1. Открываешь два терминала в VS Code.
Терминал 1 — Бэкенд (Сервер)
bash
node server.js
Должен быть в папке будущее (там где лежит server.js).

После запуска увидишь: ✅ Сервер запущен на http://localhost:3000.

Терминал 2 — Фронтенд (Сайт)
bash
cd client
npm start
Сначала переходим в папку client.

После запуска в браузере откроется вкладка с сайтом (адрес обычно http://localhost:3001).
